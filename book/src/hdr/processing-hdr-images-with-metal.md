# Processing HDR images with Metal

> [Apple Documentation](https://developer.apple.com/documentation/metal/processing-hdr-images-with-metal?language=objc)

## Run the Example

```bash
cargo run --example processing_hdr_images_with_metal
```

## Shader Files

- `Renderer/AAPLShaders.metal`

- `Assets/kloppenheim_06_4k.hdr`
