# Control the ray tracing process using intersection queries

> [Apple Documentation](https://developer.apple.com/documentation/metal/control-the-ray-tracing-process-using-intersection-queries?language=objc)

Explicitly enumerate a ray’s intersections with acceleration structures by creating an intersection query object.

## Run the Example

```bash
cargo run --example control_the_ray_tracing_process_using_intersection_queries
```

## Shader Files

- `Renderer/Shaders.metal`
