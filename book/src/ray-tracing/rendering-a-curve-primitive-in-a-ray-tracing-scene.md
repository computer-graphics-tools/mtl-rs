# Rendering a curve primitive in a ray tracing scene

> [Apple Documentation](https://developer.apple.com/documentation/metal/rendering-a-curve-primitive-in-a-ray-tracing-scene?language=objc)

## Run the Example

```bash
cargo run --example rendering_a_curve_primitive_in_a_ray_tracing_scene
```

## Shader Files

- `Renderer/Shaders.metal`
