# Accelerating ray tracing using Metal

> [Apple Documentation](https://developer.apple.com/documentation/metal/accelerating-ray-tracing-using-metal?language=objc)

## Run the Example

```bash
cargo run --example accelerating_ray_tracing_using_metal
```

## Shader Files

- `Renderer/Shaders.metal`

## Overview

Note

This sample code project is associated with WWDC22 session 10105: Maximize your Metal ray tracing performance and WWDC20 session 10012: Discover ray tracing with Metal.

Configure the sample code project

This sample requires the following system and software configuration:



Xcode 12 or later
