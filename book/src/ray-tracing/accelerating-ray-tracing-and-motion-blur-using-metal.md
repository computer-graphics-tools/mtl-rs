# Accelerating ray tracing and motion blur using Metal

> [Apple Documentation](https://developer.apple.com/documentation/metal/accelerating-ray-tracing-and-motion-blur-using-metal?language=objc)

## Run the Example

```bash
cargo run --example accelerating_ray_tracing_and_motion_blur_using_metal
```

## Shader Files

- `Renderer/Shaders.metal`

## Resources

- `Ninja/ninja.mtl`

## Overview

Note

This sample code project is associated with WWDC22 session 10105: Maximize your Metal ray tracing performance and WWDC21 session 10149: Enhance your app with Metal ray tracing.

Configure the sample code project

This sample requires the following system and software configuration:



Xcode 13 or later
