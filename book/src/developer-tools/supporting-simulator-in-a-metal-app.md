# Supporting Simulator in a Metal app

> [Apple Documentation](https://developer.apple.com/documentation/metal/supporting-simulator-in-a-metal-app?language=objc)

## Run the Example

```bash
cargo run --example supporting_simulator_in_a_metal_app
```

## Shader Files

- `Swift/Renderer/Shaders.metal`
- `Objective-C/Renderer/Shaders.metal`

## Resources

- `Swift/Renderer/Assets.xcassets/ColorMap.textureset/Universal.mipmapset/ColorMap.png`
- `Objective-C/Renderer/Assets.xcassets/ColorMap.textureset/Universal.mipmapset/ColorMap.png`
